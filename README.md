## 后端启动流程

1. ```bash
   $ git clone https://github.com/break1145/depato_lease_be.git
   $ cd depato_lease_be
   ```


2. 打开idea，更新maven仓库，直接运行默认启动类`AdminWebApplication`
3. MySQL, Redis已配置好云服务器，开箱即用
